File and their purpose.

- ANN_Adam.py : Script to train the ANN for state prediction
- ANN_test.py : Script to test the ANN for state prediction
- ANNQuantile.py : Script to train the ANN for quantile regression
- ANNQuantile_test.py : Script to test the ANN for quantile regression
- prediction.py : Script to perform the open-loop prediction of the ANN state prediction
- finalPlot.py : Script to plot and display everything on test data. Also contains conformal prediction
